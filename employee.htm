EMPLOYEE MANAGEMENT

This module of ATTendee HRM,allows the HRM manage employees and their data. It makes it possible to create employee, Activate employee, upload employee data to device in order to allow physical capture via biometrics. Many more features can be operated from this modules. The following guide shows how employee management can be done without much hassle

On your dashboard, Click on the menu at the left corner then select Employee Management



Fig 1.0 : Employee Management

It can be seen from Fig 1.0 that the following task can be perform in this module

  1. Create Employee
  2. Activate Employee
  3. Inactivate Employee
  4. Delete Employee
  5. Import Employee
  6. Upload Employee data to device
  7. Group Assigning
    8. etc.
